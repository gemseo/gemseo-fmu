    Time series

    time_series ¶

    Time series.

    Attributes¶

    ObservableType module-attribute ¶

    ObservableType = Union[
    Sequence[float], NDArrayPydantic[float]
]

    The type for a sequence of observable values.

    TimeType module-attribute ¶

    TimeType = Union[
    Sequence[TimeDurationType], NDArrayPydantic
]

    The type for a sequence of time values.

    Classes¶

    TimeSeries ¶

    The time series of an observable.

    Attributes¶
    compute class-attribute instance-attribute ¶
    compute: Callable[[float], float] = field(init=False)

    The stairs function built from this time series.

    observable instance-attribute ¶
    observable: ObservableType

    The values of the observable associated to the values of the time.

    size class-attribute instance-attribute ¶
    size: int = field(init=False)

    The size of the time series.

    time instance-attribute ¶
    time: TimeType

    The increasing values of the time.

    The components can be either numbers in seconds or strings of characters (see TimeDuration).

    tolerance class-attribute instance-attribute ¶
    tolerance: TimeDurationType = 0.0

    The tolerance for the stairs function.
